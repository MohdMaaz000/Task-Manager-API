Task Manager API

A production-style Flask REST API for authentication and task management. It provides JWT-secured endpoints for user auth, task CRUD operations, filtering, analytics, and health monitoring in a clean deployable backend.

Core Features

  • JWT authentication with access and refresh tokens
  • Protected task CRUD, filtering, and statistics
  • Validation, rate limiting, and health checks
  • Hosted deployment with Render

Tech Stack

  • Python, Flask, SQLAlchemy
  • Flask-JWT-Extended
  • Flask-Limiter and Flask-Migrate
  • Render deployment and GitHub CI

Main Endpoints

  • GET/health
  • POST/auth/register
  • POST/auth/login
  • GET/auth/me
  • GET/tasks
  • POST/tasks
  • GET/tasks/stats

Usage

Test the API with Postman or connect it to a frontend client. Start with /auth/register, then /auth/login, and use the returned bearer token for protected task routes.

Recommended public checks: / for the landing page and /health for service status.
